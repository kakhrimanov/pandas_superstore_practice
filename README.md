# Практикум Pandas на данных Superstore

Этот репозиторий содержит практические задания по анализу данных с использованием **pandas**, **matplotlib** и **seaborn** на датасете продаж Superstore.

## Структура репозитория

- `tasks.ipynb` — ноутбук с заданиями для самостоятельного выполнения  
- `tasks_solution.ipynb` — ноутбук с ответами и примерами решений  
- `Sample - Superstore.csv` — исходный датасет  

## Как работать с репозиторием

Рекомендуемый порядок работы:

1. Откройте ноутбук `tasks.ipynb`
2. Попробуйте решить задания самостоятельно
3. При необходимости воспользуйтесь подсказками
4. Если подсказка не помогает — обратитесь к `tasks_solution.ipynb`


## Требования

- Python 3.8+
- Jupyter Notebook
- pandas
- matplotlib
- seaborn
